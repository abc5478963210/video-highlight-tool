# Video Highlight Tool

> AI 影片精華擷取工具  
> 風格參考 [GliaCloud](https://www.gliacloud.com/en/) —— 深色主題、淡紫主色、科技感漸層裝飾

![screenshot](./screenshot.png)

## 介面設計特色
- 深色背景、淡紫主色
- 漸層幾何裝飾
- 大標題、圓角按鈕
- 響應式設計

## 快速啟動

```bash
npm install
npm run dev
```

## 主要技術
- Vue 3 + TypeScript
- Ant Design Vue
- SCSS（含漸層、毛玻璃、全域變數）

## 專案結構
- `src/styles/`：全域 SCSS 變數、混入、主樣式
- `src/views/`：主要頁面元件
- `src/components/`：可複用元件
